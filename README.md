# 修仙传奇：天命之旅

一款基于React和TypeScript开发的修仙主题文字冒险游戏。玩家将扮演一位修仙者，在游戏中通过各种选择塑造自己的修仙之路，提升修为，经历各种奇遇和挑战。

## 游戏特色

- **丰富的剧情选择** - 多条主线分支，每个选择都会影响后续剧情走向
- **独特的修仙境界系统** - 从凡人到仙人的完整修仙之路，包含练气、筑基、结丹、元婴、化神、大乘等境界
- **角色属性培养系统** - 魅力、悟性、体质、气运、家境五大属性，影响剧情走向和选择
- **完善的成就系统** - 30+成就等待解锁，涵盖入门、道路、事件、幸运、情感、探索、结局等多个类别
- **多种小游戏交互环节** - 炼丹、火候控制、时机把握、心魔考验、九九雷劫等小游戏
- **精美的视觉效果和动画** - 流畅的动画效果，精美的背景图片，沉浸式游戏体验
- **完整的存档/读档功能** - 随时保存游戏进度，支持多个存档槽位
- **剧情回顾系统** - 查看已解锁的剧情节点，了解游戏进度
- **多结局系统** - 飞升、至尊、守护、传奇、陨落等多种结局，探索不同的修仙之路
